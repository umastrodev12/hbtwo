## ❓ O que é o HobbyTwo?
### 😃 Explicação Simples
O HobbyTwo é um aplicativo feito para hobby's e gostos diferentes de cada usuário.
### 📚 Explicação Detalhada
Definitivamente, eu tive essa ideia enquanto estava navegando pelo GitHub, aí eu fui atrás de ideias para fazer um "simples aplicativo", até que eu achei, O HobbyTwo basicamente é Quase um Chat, mas a ideia principal não é um chat.

Eu definitivamente só tive a ideia e fiz esse app só por diversão mesmo :D
### 🤔 Por quê Hobby's?
Sabe, hoje em dia não existe aplicativos especificados para "Hobby's e Gostos de Usuários", então decidi criar o meu.




## 🌐 Features
 - Criação de Comunidades para cada Hobby <sup>Beta</sup>
 - Um website para este aplicativo <sup>Pré-Lançamento</sup>
